# Archive Index

## Purpose

This folder contains historical planning artifacts that are no longer authoritative.

Archived docs are kept for provenance and project memory, but they should not be used as the source of truth for active implementation.

## Archive contents

### `letters/`

Historical vision briefs and early prompts that helped establish the system direction.

### `educational-design-system/`

Superseded planning or repair artifacts whose contents have been integrated into the active spec set.

## Usage rule

Only consult archived docs when you need historical context, decision lineage, or original framing.

For active implementation, return to `docs/_specs/README.md` and the active docs in `docs/_specs/educational-design-system/`.
